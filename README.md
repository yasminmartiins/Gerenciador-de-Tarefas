# Gerenciador-de-Tarefas

## O que é:

A aplicação é um CRUD (Create, Read, Update, Delete) de um gerenciador de tarefas divido em 4 telas:

- Menu Principal
- Criar Tarefa
- Status da Tarefa (Contém "Deletar" e "Buscar")
- Editar Tarefa

## Tecnologias utilizadas:

A aplicação teve o back-end feito em python com framework Flask, um banco de dados local (SQLite) e o front-end foi feito em HTML, CSS e JavaScript.

## Como executar:

Executar no terminal da aplicação os seguintes comandos:

- pip install flask
- python database.py (Caso o banco de dados não tenha sido executado)
- python main.py

Após a execução um link será gerado e deve ser aberto no navegador:

http://127.0.0.1:5000
